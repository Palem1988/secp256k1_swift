# secp256k1-swift

A description of this package.

```
$ git clone https://github.com/bitcoin-core/secp256k1.git
$ cd secp256k1
$ ./autogen.sh
$ ./configure
$ make
$ ./tests
$ sudo make install  # optional

```

then 

```
.package(url: "https://github.com/TwicebookSwift/secp256k1_swift", from:"1.0.0")
```
